Github Action Openwrt固件自动编译教程

视频教程

1:根据L大的源码仓库指导步骤，利用 Ubuntu 18 LTS x64的VPS获取对应路由器型号的.config文件。

2:利用的源码仓P3TERX导入.config文件启动GitHub Action自动云编译。

3:进入 P3TERX/Actions-OpenWrt 项目页面，点击 Use this template 。

4:填写仓库名称，然后点击Create repository from template按钮。经过几秒钟的等待，页面会跳转到新建的仓库，然后点击Create new file按钮。
5：然后文件名填写为.config，把生成的.config 文件的内容复制粘贴到下面的文本框中.

最后等待2-3小时前往Github Action下载压缩包固件使用即可。
